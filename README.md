# Bezier Curve Editor 贝塞尔曲线编辑器

## 代码功能

这是一个基于PyGame的贝塞尔曲线可视化编辑器。主要功能包括：

1. **多种模式切换**：
   - 创建模式：添加、删除、拖拽控制点
   - 递归构造模式：展示De Casteljau算法过程
   - 向量表示模式：展示Bernstein基函数的向量合成
   - 动力学分析模式：显示速度、加速度等导数向量
   - 3D演示模式：将2D曲线扩展到3D空间显示

2. **实时交互**：支持鼠标拖拽控制点、缩放平移视图

3. **参数调整**：通过滑块实时调整参数t值

## 技术特点

- 使用PyGame进行图形渲染
- 实现了De Casteljau算法
- 支持Bernstein基函数计算和可视化
- 实现了3D投影和坐标变换
- 模块化设计，便于扩展

## 用户体验

- 中文界面，操作提示明确
- 可拖拽的控制面板
- 音效和背景音乐
- 详细的帮助系统
- 实时状态显示

## 安装要求（仅限Windows系统）

### 系统要求
- Windows 7/10/11
- Python 3.8或更高版本

### 安装步骤
1. 安装Python：从python.org下载并安装
2. 安装PyGame：
   ```bash
   pip install pygame
   ```
3. 下载项目文件，确保包含以下目录结构：
   ```
   bezier_curve_editor/
   ├── main.py
   ├── config.py
   ├── resources/（包含图标、音效等资源）
   └── src/（包含算法模块）
   ```

4. 运行程序：
   ```bash
   python main.py
   ```

## 快速开始

1. 运行程序后，默认处于创建模式
2. 在窗口中左键点击添加控制点（至少需要2个点）
3. 自动生成贝塞尔曲线
4. 按数字键切换不同模式：
   - 1：创建模式
   - 2：递归构造模式
   - 3：向量表示模式
   - 4：动力学分析模式
   - 5：3D演示模式

## 技术实现

### 核心算法
- 贝塞尔曲线计算：基于Bernstein多项式
- 递归构造：De Casteljau算法实现
- 导数计算：速度、加速度、急动度向量
- 3D投影：将3D坐标转换为2D屏幕坐标

### 界面系统
- 可拖拽面板系统
- 统一的缩放和平移管理
- 事件处理机制

## 视觉效果

- 控制点：红色圆点
- 控制线：灰色虚线
- 贝塞尔曲线：绿色实线
- 向量：彩色箭头
- 3D显示：RGB颜色映射

## 注意事项

1. 运行前请关闭中文输入法，否则快捷键可能失效
2. 确保resources目录存在且包含必要的资源文件
3. 3D演示模式需要较多计算资源，性能较差的电脑可能会有卡顿
4. 程序仅支持Windows系统
